# Loan Application

Тестовое React-приложение для интервью в компанию WIAM

## Технологии

- React
- TypeScript
- React Router
- React Hook Form — было использовано так как удобная работа с формами, минимизация перерисовок и встроенная валидация
- IMask — использовал чтобы не тратить время на написание маски ввода для поля телефона
- Bootstrap

## Запуск проекта

1. Клонировать репозиторий

```bash
git clone https://github.com/YOUR_USERNAME/loan-application-form.git
```

2. Перейти в папку проекта

```bash
cd loan-form-app
```

3. Установить зависимости

```bash
npm install
```

4. Запустить проект

```bash
npm run dev
```

После запуска приложение будет доступно по адресу:

```
http://localhost:5173
```
